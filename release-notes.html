Shiori 1.0.0 – The Official Launch!

April 2026

What's New

  • Link Groups: Save a set of links and open them all at once from the widget!
  • Chrome Profile Routing: Send specific URLs to specific Chrome profiles based on rules you set!
  • Search Bars: Filter bookmarks, folders, tags, and presets without leaving the section you're in!

Improvements

  • Widget: Navigate into Link Groups straight from the widget and launch the whole set!
  • UI Polish: Standardized buttons throughout, tidier menus, and cleaner text across the app!

Bug Fixes

  • Light Mode: Fixed light mode rendering issues!
  • Login Screen: Fixed broken layout and sign-in flow issues!

Thank you for being part of the Shiori community! Your feedback drives every improvement we make – keep it coming!
